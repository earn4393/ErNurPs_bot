# ErNurPs_bot
 Project นี้เป็นหนึ่งในวิชา Exploratory Project in Computing

 โดยการแนะนำ มีดังนี้
  - เกรดที่คาดว่าจะได้
  - แจ้งเตือนการบ้านที่ต้องส่ง
  - คุยเล่น
